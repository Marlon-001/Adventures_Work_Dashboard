# Adventures_Work_Dashboard

### Power BI Training Project | Maven Analytics

I developed this **Executive Sales Dashboard** using the Microsoft **AdventureWorks** dataset to transform raw manufacturing and sales data into a strategic tool for business decision-making.

---

## 🚀 Project Overview
The dashboard provides a comprehensive view of business health, tracking **$24.9M** in total revenue and identifying key trends across product categories and timeframes.

## 📈 Key Insights & Features
* **Executive KPIs:** Instant visibility into **Total Revenue ($24.9M)**, **Profit ($10.5M)**, and **Total Orders (25.2K)**.
* **Revenue Trends:** Weekly line charts with trend analysis to monitor growth and seasonality.
* **Product Performance:** Detailed matrix highlighting return rates and revenue per product, using **Conditional Formatting** to flag high-return items.
* **Category Analysis:** Breakdown of sales volume by category (Accessories, Bikes, and Clothing) to understand product mix.
* **Time Intelligence:** Monthly revenue cards tracking a **3.31% increase** in performance vs. the previous month.

## 🛠️ Technical Skills Applied
* **Data Modeling:** Connecting and structuring the AdventureWorks relational database.
* **DAX (Data Analysis Expressions):** Developed measures for dynamic KPIs and Month-over-Month (MoM) comparisons.
* **Data Visualization:** Designed a clean UI/UX with intuitive navigation and color-coded alerts.
* **Power Query:** Cleaned and transformed raw data for analysis.
